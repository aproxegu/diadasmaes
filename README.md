# Aproxegu — Quiz de Essências

Sistema de quiz personalizado para a [Aproxegu](https://aproxegu.com.br), com captura de leads e painel administrativo.

## Páginas

| Página | Descrição |
|--------|-----------|
| `index.html` | Quiz principal — Descubra sua essência |
| `dia-das-maes.html` | Quiz Dia das Mães — Edição Especial |
| `admin.html` | Painel administrativo (acesso restrito) |

## Como usar

Os arquivos são HTML puro — basta hospedar em qualquer serviço de hospedagem estática (GitHub Pages, Netlify, Vercel).

## Acesso ao painel

O painel admin está disponível em `/admin.html`. Acesso por senha.
